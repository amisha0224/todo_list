# Todo List App

A simple todo list app built with React and Vite. The app keeps the UI clean and focused on basic CRUD operations:

- Create a task
- Read the task list
- Update a task inline
- Delete a task
- Mark tasks as completed or pending

## Tech Stack

- React
- Vite
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Structure

```text
src/
  App.jsx
  App.css
  index.css
```

## Notes

- `node_modules` and `dist` are ignored by Git.
- This project uses local React state and does not require a backend.
